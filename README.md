# JavaExplore
======
## 项目概要
    1.本项目用于java基础及进阶学习，并以java web形式扩展。
    2.本项目使用IDEA 2016开发，使用maven构建,使用Spring Boot与MyBatis，数据库使用MySQL5.6+
    3.本项目下包含若干子项目，侧重于学习不同方向
### java-explore-basic
   ####简介
    * 用于学习java基础及进阶知识
    * 用于学习及研究java源码知识
### java-explore-web
   ####简介
    * 用于实践java学习部分
    * 用于学习实践web项目相关知识
    * 用于学习实践js相关知识
